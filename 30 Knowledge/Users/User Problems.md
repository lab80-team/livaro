# User Problems

## Şu An Bilinenler
- **Kurucuların problem hipotezi netleşti (2026-07-24 thinking session)** — dış kullanıcıyla henüz doğrulanmadı (**Needs Validation**):
  1. Evini dizmek/mobilya yenilemek isteyen kişi az bildiği bir sektörü araştırmak zorunda; ya mimarla çalışıyor ya kendi başına deniyor.
  2. Mobilya, halı, perde, avize **ayrı ayrı mağazalardan** alınıyor; mağaza gezme süreci **1-2 ay** sürüyor.
  3. Ürünler satın alınmadan **odada bir arada görülemiyor** → karar vermek zorlaşıyor.
  4. Profesyonel tasarım desteği alınmadığı için odanın potansiyeli gerçekleşmiyor.
  5. Farklı mağazalardan alım → **toplam bütçe tek yerde görünmüyor**, ürün başına bütçe dağılımı baştaki tahminle uyuşmuyor → bütçe aşılıyor.
  6. Sonuç: zaman kaybı + fiziksel/mental yorgunluk.
